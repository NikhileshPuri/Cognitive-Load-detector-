# Cognitive-Load-detector-
Cognitive load detector based on the COGBCI dataset!

EEG-based cognitive load classification using Riemannian geometry (Tangent Space), 
covariance matrices, and machine learning. Pipeline: preprocessing, sliding-window 
epoching, band-specific covariance extraction, tangent space embedding, 
classification (LR/RF/XGBoost), prediction smoothing, and real-time playback 
visualization. Includes SHAP explainability and LOSO evaluation.
